# AI Agent實戰課程

 針對金融業場景的 AI Agent 開發實戰訓練，從基礎概念到企業級部署的完整課程。

## Day 1: AI Agent 導入與語音整合

AI Agent 核心概念與金融業應用場景
LangChain Agent 實作
OpenAI Whisper 語音整合與金融術語優化
LangGraph 語音流程：STT → 詞彙修正 → 回答生成

## Day 2: Tools 開發與 MCP 整合


LangGraph 多分支編排：意圖分類 → 動態路由（股價/投資組合/問答）
LangGraph 進階：記憶管理、Human-in-the-Loop（風險審核流程）
MCP (Model Context Protocol) 標準與實作


## Day 3: Gateway 架構與企業部署

企業級 MCP Gateway 設計（多 Server 管理、Routing）
LangGraph 在多模型協調中的應用
效能優化（Redis、Connection Pooling、Asyncio）
微服務架構與 Docker 容器化
監控系統（Prometheus + Grafana）
